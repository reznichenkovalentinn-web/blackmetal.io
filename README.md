# BlackMetal

Готовий багатосторінковий сайт на HTML5, CSS3, JavaScript і невеликому PHP-обробнику для Telegram-заявок.

## Запуск

Для звичайного перегляду відкрийте `index.html`. Для роботи форми потрібен хостинг із PHP 8+ та HTTPS.

## Підключення Telegram-бота

1. Створіть бота через `@BotFather` і отримайте токен.
2. Напишіть повідомлення вашому боту та визначте ID чату через Telegram Bot API.
3. Скопіюйте `telegram-config.example.php` як `telegram-config.php`.
4. Вставте токен бота та ID чату у `telegram-config.php`.
5. Не публікуйте файл із токеном у відкритому репозиторії.

Секретний токен навмисно не зберігається в JavaScript — інакше його зможе побачити будь-який відвідувач.

## Домен і SEO

Після підключення домену замініть `https://example.com` у `sitemap.xml` на адресу сайту та додайте повне посилання на sitemap у `robots.txt`.

## Додавання робіт

Додайте фото до `assets/images`, скопіюйте одну кнопку `.gallery-item` у `works.html` і вкажіть категорію `gates` або `canopies`. Лайтбокс та фільтрація підключаться автоматично.
